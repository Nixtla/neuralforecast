This file provides a set of unit tests for all models

------------------------------------------------------------------------

### check_model

> ``` text
>  check_model (model_class, checks=['losses', 'airpassengers'])
> ```

*Check model with various tests. Options for checks are:<br/> “losses”:
test the model against all loss functions<br/> “airpassengers”: test the
model against the airpassengers dataset for forecasting and
cross-validation<br/>*
